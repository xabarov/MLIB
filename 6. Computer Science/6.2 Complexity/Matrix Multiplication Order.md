# Порядок умножения матриц

**Вопрос (EN)**: A∈R^{100×5}, B∈R^{5×200}, C∈R^{200×20}. Order for ABC? For N matrices A1...An?

## Ответ

### Порядок для ABC

Число операций (умножений) для (m×n)(n×p) = m×n×p.

- (AB)C: 100×5×200 + 100×200×20 = 100,000 + 400,000 = 500,000
- A(BC): 5×200×20 + 100×5×20 = 20,000 + 10,000 = 30,000

Оптимально **A(BC)** — 30,000 против 500,000.

### N матриц

Задача — найти порядок умножения с минимальным числом операций. Решается динамическим программированием: \(m[i,j]\) — минимальная стоимость перемножения \(A_i \ldots A_j\). Базовый случай \(m[i,i]=0\); переход — перебор места разбиения \(k\).
