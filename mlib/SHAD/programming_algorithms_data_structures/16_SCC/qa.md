## Задача 1. Определение SCC вручную

Дан орграф на 6 вершинах с рёбрами:
$0 \to 1$, $1 \to 2$, $2 \to 0$, $3 \to 2$, $3 \to 4$, $4 \to 5$, $5 \to 3$.

Найдите все сильно связные компоненты. Постройте конденсацию.

### Решение

Шаг 1. Проверим взаимодостижимость.

- $0 \to 1 \to 2 \to 0$: цикл, значит $\{0,1,2\}$ — одна SCC. Назовём её $A$.
- $3 \to 4 \to 5 \to 3$: цикл, значит $\{3,4,5\}$ — одна SCC. Назовём её $B$.
- Из $B$ можно достичь $A$: $3 \to 2 \in A$. Из $A$ нет пути в $B$ (ни одно ребро не ведёт в $\{3,4,5\}$). Значит, $A$ и $B$ — разные SCC.

Итого: **две SCC** — $A = \{0,1,2\}$ и $B = \{3,4,5\}$.

Шаг 2. Конденсация.

Ребро $3 \to 2$ (из $B$ в $A$): в конденсации $B \to A$.

Конденсация: два узла $A$ и $B$, одно ребро $B \to A$. Это DAG.

### Ответ

SCC: $A = \{0,1,2\}$, $B = \{3,4,5\}$. Конденсация: $B \to A$.

## Задача 2. Тривиальные компоненты

Дан орграф: $0 \to 1$, $1 \to 2$, $2 \to 3$, $3 \to 0$, $1 \to 4$, $4 \to 5$.

Сколько SCC в этом графе? Какие вершины образуют нетривиальные SCC?

### Решение

Шаг 1. Ищем циклы.

- $0 \to 1 \to 2 \to 3 \to 0$: цикл, все четыре вершины в одной SCC: $C_1 = \{0,1,2,3\}$.
- Вершина 4: есть ли путь $4 \to \cdots \to 4$? Из 4 идём в 5, из 5 нет исходящих рёбер. Нет цикла. SCC: $C_2 = \{4\}$.
- Вершина 5: нет исходящих рёбер. SCC: $C_3 = \{5\}$.

Шаг 2. Конденсация.

Ребро $1 \to 4$: $C_1 \to C_2$. Ребро $4 \to 5$: $C_2 \to C_3$.
Конденсация: $C_1 \to C_2 \to C_3$.

### Ответ

Три SCC: $\{0,1,2,3\}$ (нетривиальная), $\{4\}$, $\{5\}$. Конденсация — путь из трёх узлов.

## Задача 3. Граф без рёбер

Рассмотрим орграф с $n = 5$ вершинами и $0$ рёбрами. Сколько SCC? Какова конденсация?

### Решение

Без рёбер из вершины $v$ нет пути ни в одну другую вершину. Отношение сильной связности вырождается: $u \sim v \Leftrightarrow u = v$ (только рефлексивные пары).

Каждая вершина — отдельная тривиальная SCC.

Конденсация: 5 изолированных узлов (DAG без рёбер).

### Ответ

5 SCC: $\{0\}$, $\{1\}$, $\{2\}$, $\{3\}$, $\{4\}$.

## Задача 4. Трассировка алгоритма Косарайю

Дан орграф: вершины $\{0,1,2,3\}$, рёбра: $0 \to 1$, $1 \to 2$, $2 \to 0$, $1 \to 3$.

Запустите алгоритм Косарайю. Опишите порядок завершения (проход 1) и DFS на $G^T$ (проход 2). Найдите SCC.

### Решение

**Граф G:**
- 0: [1]; 1: [2,3]; 2: [0]; 3: []

**Транспонированный $G^T$:**
- 0: [2]; 1: [0]; 2: [1]; 3: [1]

**Проход 1 (DFS на G, стартуем с 0):**

```
dfs(0): посещаем 0
  dfs(1): посещаем 1
    dfs(2): посещаем 2
      ребро 2→0: 0 уже посещена, пропускаем
    exit(2): order = [2]
    dfs(3): посещаем 3
      нет исходящих рёбер
    exit(3): order = [2, 3]
  exit(1): order = [2, 3, 1]
exit(0): order = [2, 3, 1, 0]
```

Стек (сверху) после прохода 1: 0, 1, 3, 2.

**Проход 2 (DFS на $G^T$, берём с вершины стека):**

- Берём 0: `dfs_T(0)`: из 0 идём по $G^T$ → 2 → 1 → 0 (цикл). SCC₀ = {0, 1, 2}.
  - Из 1 в $G^T$: соседи 0 (посещена), нет новых. Выходим.
  - Из 2 в $G^T$: соседи 1 (посещена). Выходим.

- Берём 1: уже посещена, пропускаем.
- Берём 3: не посещена. `dfs_T(3)`: из 3 в $G^T$ — сосед 1 (посещена). SCC₁ = {3}.
- Берём 2: уже посещена.

**Результат:** SCC₀ = {0,1,2}, SCC₁ = {3}.

### Ответ

Две SCC: $\{0,1,2\}$ и $\{3\}$. Порядок завершения: 2, 3, 1, 0.

## Задача 5. Трассировка алгоритма Тарьяна

Используя тот же граф из задачи 4 ($0 \to 1$, $1 \to 2$, $2 \to 0$, $1 \to 3$), запустите алгоритм Тарьяна. Заполните таблицу disc/low для каждой вершины и укажите момент выталкивания SCC.

### Решение

Инициализация: timer = 0, стек = [].

**dfs(0):**
- disc[0]=0, low[0]=0, стек=[0]

  **dfs(1):**
  - disc[1]=1, low[1]=1, стек=[0,1]

    **dfs(2):**
    - disc[2]=2, low[2]=2, стек=[0,1,2]
    - ребро 2→0: onStack[0]=true → low[2]=min(2,disc[0])=min(2,0)=**0**
    - low[2]=0 ≠ disc[2]=2 → не корень

  - Возврат из dfs(2): low[1]=min(1,low[2])=min(1,0)=**0**

    **dfs(3):**
    - disc[3]=3, low[3]=3, стек=[0,1,2,3]
    - нет исходящих рёбер
    - low[3]=3 == disc[3]=3 → **корень SCC!**
    - Выталкиваем: {3} → **SCC₀ = {3}**

  - Возврат из dfs(3): low[1]=min(0,low[3])=min(0,3)=**0**
  - low[1]=0 ≠ disc[1]=1 → не корень

- Возврат из dfs(1): low[0]=min(0,low[1])=min(0,0)=**0**
- low[0]=0 == disc[0]=0 → **корень SCC!**
- Выталкиваем: 2, 1, 0 → **SCC₁ = {0,1,2}**

**Итоговая таблица:**

| Вершина | disc | low | SCC |
|---------|------|-----|-----|
| 0 | 0 | 0 | 1 |
| 1 | 1 | 0 | 1 |
| 2 | 2 | 0 | 1 |
| 3 | 3 | 3 | 0 |

### Ответ

SCC: $\{3\}$ (корень при disc=low=3), $\{0,1,2\}$ (корень вершина 0, low=disc=0).

## Задача 6. Подсчёт рёбер в конденсации

Дан орграф с $n$ вершинами и $m$ рёбрами. Докажите, что количество рёбер в конденсации не превышает $m$.

### Решение

Каждое ребро конденсации $SCC_i \to SCC_j$ (с $i \neq j$) порождено хотя бы одним ребром оригинального графа $u \to v$ ($u \in SCC_i$, $v \in SCC_j$). Это ребро оригинального графа — единственное «свидетельство» данного ребра конденсации.

Построим отображение: сопоставим каждому ребру конденсации одно ребро оригинального графа (любое из тех, что его породило). Это отображение инъективно: разные рёбра конденсации $SCC_i \to SCC_j$ и $SCC_k \to SCC_l$ (где $(i,j) \neq (k,l)$) имеют разные пары компонент-концов, значит, ни одно ребро оригинального графа не может одновременно порождать оба. Следовательно, число рёбер конденсации не превышает числа рёбер оригинального графа $m$.

### Ответ

Инъекция рёбер конденсации в рёбра оригинального графа гарантирует, что $|E_{cond}| \leq m$.

## Задача 7. Поиск источника конденсации

По данному орграфу $G(V, E)$ найдите за $O(V + E)$ хотя бы одну SCC, являющуюся *источником* конденсации (SCC с нулевой степенью входа в DAG конденсации).

Предложите алгоритм без явного построения конденсации.

### Решение

**Наблюдение:** В алгоритме Косарайю SCC, обрабатываемая последней в проходе 1 (вершина с максимальным $t_{out}$), является источником конденсации.

**Алгоритм:**

1. Запустить DFS на $G$. Найти вершину $v^*$ с максимальным временем завершения $t_{out}$.
2. $v^*$ принадлежит SCC-источнику. Эта SCC — корень соответствующего дерева DFS в проходе 1.

**Обоснование.** Вершина $v^*$ с максимальным $t_{out}$ завершается позже всех. Это значит, что в момент запуска DFS из $v^*$ ещё не было рассмотрено ни одного ребра из другой SCC, которая могла бы «завершиться позже» (иначе она бы имела больший $t_{out}$). Значит, из $SCC(v^*)$ нет рёбер, входящих из других SCC — она является источником.

**Реализация:** один DFS, запоминаем последнюю добавленную в `order` вершину. Её компонента — источник.

### Ответ

Вершина с максимальным $t_{out}$ принадлежит SCC-источнику конденсации. Находится за $O(V+E)$.

## Задача 8. Применение к 2-SAT

Дана формула 2-SAT: $(x_1 \lor \lnot x_2) \land (\lnot x_1 \lor x_3) \land (x_2 \lor x_3)$.

Постройте импликационный граф, найдите SCC, определите выполнимость и, если выполнима, найдите выполняющее присвоение.

### Решение

**Шаг 1. Вершины импликационного графа:**
- $x_1$ (вершина 0), $\lnot x_1$ (вершина 1), $x_2$ (2), $\lnot x_2$ (3), $x_3$ (4), $\lnot x_3$ (5).

**Шаг 2. Рёбра (клауза $a \lor b$ → рёбра $\lnot a \to b$ и $\lnot b \to a$):**

- $(x_1 \lor \lnot x_2)$: $\lnot x_1 \to \lnot x_2$ и $x_2 \to x_1$.
  - Рёбра: $1 \to 3$, $2 \to 0$.
- $(\lnot x_1 \lor x_3)$: $x_1 \to x_3$ и $\lnot x_3 \to \lnot x_1$.
  - Рёбра: $0 \to 4$, $5 \to 1$.
- $(x_2 \lor x_3)$: $\lnot x_2 \to x_3$ и $\lnot x_3 \to \lnot x_2$.
  - Рёбра: $3 \to 4$, $5 \to 3$.

**Шаг 3. Полный граф:**
- 0→4, 1→3, 2→0, 3→4, 5→1, 5→3.

**Шаг 4. Найдём SCC** (вручную):
- Проверяем циклы. Из 0 идём: 0→4. Из 4 нет исходящих. Нет цикла.
- Все рёбра ведут «в одну сторону» (к вершинам 4,3,1,0). Нет обратных путей.
- SCC: {0}, {1}, {2}, {3}, {4}, {5} — все тривиальны.

**Шаг 5. Проверка выполнимости:**
Для каждой переменной $x_i$: если $x_i$ и $\lnot x_i$ в одной SCC → невыполнима. Здесь — все SCC тривиальны, пар нет → **выполнима**.

**Шаг 6. Присвоение значений:**
В топологическом порядке конденсации: SCC с $\lnot x_i$ раньше $x_i$ → $x_i = \text{true}$.

Топологический порядок (по $t_{out}$ в обратном направлении): 5→1→3→2→0→4.

- $x_1$: вершины 0 ($x_1$) и 1 ($\lnot x_1$). В порядке 1 идёт раньше 0 → $x_1 = \text{true}$.
- $x_2$: вершины 2 ($x_2$) и 3 ($\lnot x_2$). В порядке 3 идёт раньше 2 → $x_2 = \text{true}$.
- $x_3$: вершины 4 ($x_3$) и 5 ($\lnot x_3$). В порядке 5 идёт раньше 4 → $x_3 = \text{true}$.

**Проверка:** $(1 \lor 0) \land (0 \lor 1) \land (1 \lor 1) = 1 \land 1 \land 1 = 1$. Верно!

### Ответ

Формула выполнима. Одно решение: $x_1 = x_2 = x_3 = \text{true}$.

## Задача 9. Подсчёт вершин, достижимых из источников

Дан орграф $G$. За $O(V + E)$ найдите количество вершин, достижимых из любого источника конденсации (SCC с нулевой входящей степенью).

### Решение

**Алгоритм:**

1. Найти все SCC (Тарьян/Косарайю). Построить конденсацию.

2. В конденсации найти все источники (SCC с $indegree = 0$).

3. Запустить BFS/DFS из всех источников по конденсации одновременно. Отметить все достижимые SCC.

4. Подсчитать суммарное количество вершин в отмеченных SCC.

**Детали построения:**
```cpp
// После нахождения SCC (comp[] заполнен)
vector<int> scc_size(num_scc, 0);
vector<int> indegree(num_scc, 0);
set<pair<int,int>> cond_edges;
vector<vector<int>> dag(num_scc);

for (int v = 0; v < n; v++) scc_size[comp[v]]++;

for (int v = 0; v < n; v++) {
    for (int u : g[v]) {
        if (comp[v] != comp[u] && cond_edges.insert({comp[v],comp[u]}).second) {
            dag[comp[v]].push_back(comp[u]);
            indegree[comp[u]]++;
        }
    }
}

// BFS от всех источников
queue<int> q;
vector<bool> visited(num_scc, false);
for (int i = 0; i < num_scc; i++) {
    if (indegree[i] == 0) { q.push(i); visited[i] = true; }
}

int total = 0;
while (!q.empty()) {
    int c = q.front(); q.pop();
    total += scc_size[c];
    for (int nb : dag[c]) {
        if (!visited[nb]) { visited[nb] = true; q.push(nb); }
    }
}
```

**Замечание:** если конденсация имеет единственный источник, это равносильно нахождению всех вершин, достижимых из данного источника в исходном графе.

### Ответ

Алгоритм за $O(V+E)$: найти SCC, построить конденсацию, BFS от источников, суммировать размеры достигнутых SCC.

## Задача 10. Минимальное число рёбер для превращения графа в сильно связный

Дан орграф $G$. Найдите минимальное количество рёбер, которые нужно добавить, чтобы граф стал сильно связным.

### Решение

Обозначим $k$ = число SCC в конденсации. Если $k = 1$ — граф уже сильно связен, ответ 0.

При $k > 1$ рассмотрим конденсацию — DAG из $k$ узлов. Нам нужно сделать его сильно связным, т.е. «замкнуть» все SCC. Для этого нужно добавить рёбра, превращающие DAG в один цикл.

**Ключевое наблюдение:**
- Пусть $s$ = число источников (SCC с $indegree_{cond} = 0$).
- Пусть $t$ = число стоков (SCC с $outdegree_{cond} = 0$).

Ответ: $\max(s, t)$.

**Обоснование:**
- Каждый источник надо «подключить»: нужно хотя бы одно ребро, входящее в него. Это $s$ рёбер.
- Каждый сток надо «подключить»: нужно хотя бы одно ребро, исходящее из него. Это $t$ рёбер.
- Одно добавленное ребро может одновременно «закрыть» один сток и один источник. Поэтому минимум — $\max(s, t)$.

**Как добавлять рёбра:**
Строим цепочку: сток₁ → источник₁ → ... → сток₂ → источник₂ → ... → сток₁. Это $\max(s,t)$ рёбер.

**Исключение:** если конденсация состоит из одного узла ($k=1$), ответ 0 без каких-либо вычислений.

**Пример:** конденсация A→B←C (два источника A,C, один сток B). $s=2$, $t=1$. Ответ: $\max(2,1)=2$. Добавляем: B→A, A→C (или B→C, C→A).

### Ответ

Минимальное число рёбер = $\max(s, t)$, где $s$ — число источников конденсации, $t$ — число стоков конденсации. Исключение: если граф уже сильно связен, ответ 0.
