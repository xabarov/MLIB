# 10 задач в стиле вступительного экзамена ШАД

по теме: Метод двух указателей. Префиксные суммы. Обобщение на произвольную обратимую операцию.

Задачи расположены по нарастающей сложности: лёгкие (1–3) закрепляют базовую технику, средние (4–6) требуют комбинирования инструментов, сложные (7–9) предполагают нетривиальное применение или доказательство, синтетическая задача (10) объединяет все темы лекции.

## Задача 1. Пара с целевой суммой

**Условие.** Дан отсортированный массив из $n$ различных целых чисел $A[0..n-1]$ и число $S$. Определите, существует ли пара индексов $i < j$ такая, что $A[i] + A[j] = S$. Если да — выведите саму пару чисел.

### Решение

Используем два указателя, сходящихся к центру.

1. Инициализируем $l = 0$, $r = n - 1$.
2. Пока $l < r$: вычисляем $t = A[l] + A[r]$.
   - Если $t = S$ — пара найдена.
   - Если $t < S$ — увеличиваем $l$ (нужна большая сумма).
   - Если $t > S$ — уменьшаем $r$ (нужна меньшая сумма).
3. Если $l \ge r$ — пары нет.

Корректность: в каждый момент все кандидаты, которые мы пропускаем, заведомо не могут дать сумму $S$. Если $A[l] + A[r] < S$, то для любого $r' < r$ выполняется $A[l] + A[r'] < S$ — такие $r'$ бесполезны, двигаем $l$.

Сложность: $O(n)$ времени, $O(1)$ памяти.

### Ответ

Алгоритм двух указателей за $O(n)$. На массиве $[1, 2, 3, 4, 5, 6, 7]$, $S = 9$: пара $(2, 7)$.

## Задача 2. Число подотрезков с суммой <= K

**Условие.** Дан массив из $n$ **неотрицательных** целых чисел $A$. Посчитайте количество подотрезков $[l, r]$ ($0 \le l \le r < n$), для которых $A[l] + \ldots + A[r] \le K$.

### Решение

Для фиксированного $l$ пусть $r^*(l)$ — наибольшее $r$ такое, что сумма $[l, r] \le K$. Тогда ровно $r^*(l) - l + 1$ подотрезков, начинающихся с $l$, удовлетворяют условию.

Из неотрицательности: при увеличении $l$ на 1 значение $r^*(l)$ не уменьшается. Ведём скользящее окно:

1. $l = 0$, $r = -1$, $\text{cur} = 0$, $\text{ans} = 0$.
2. Для каждого $l$ от 0 до $n-1$:
   a. Пока $r + 1 < n$ и $\text{cur} + A[r+1] \le K$: $r \gets r + 1$, $\text{cur} \mathrel{+}= A[r]$.
   b. $\text{ans} \mathrel{+}= r - l + 1$ (число допустимых $r$ для данного $l$).
   c. $\text{cur} \mathrel{-}= A[l]$.

Суммарно $r$ делает не более $n$ шагов вправо. Итого $O(n)$.

### Ответ

Ответ: $\sum_{l=0}^{n-1} \max(0, r^*(l) - l + 1)$, вычисляется за $O(n)$.

## Задача 3. Суммы подотрезков за O(1)

**Условие.** Дан массив $A = [2, 4, -3, 5, 1, -2, 7]$. Постройте массив префиксных сумм $P$ и ответьте на следующие запросы:
- $\text{sum}[0, 2]$
- $\text{sum}[2, 5]$
- $\text{sum}[1, 6]$

### Решение

Вычисляем $P[0] = 0$, $P[i] = P[i-1] + A[i-1]$:

| $i$ | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|---|---|---|---|---|---|---|---|---|
| $P[i]$ | 0 | 2 | 6 | 3 | 8 | 9 | 7 | 14 |

Формула: $\text{sum}[l, r] = P[r+1] - P[l]$.

- $\text{sum}[0, 2] = P[3] - P[0] = 3 - 0 = 3$.
- $\text{sum}[2, 5] = P[6] - P[2] = 7 - 6 = 1$.
- $\text{sum}[1, 6] = P[7] - P[1] = 14 - 2 = 12$.

### Ответ

$\text{sum}[0,2] = 3$, $\quad \text{sum}[2,5] = 1$, $\quad \text{sum}[1,6] = 12$.

## Задача 4. Длиннейший подотрезок с суммой равной K

**Условие.** Дан массив $A$ из $n$ целых чисел (могут быть отрицательные и нули) и число $K$. Найдите длину наибольшего подотрезка с суммой ровно $K$.

### Решение

Наивный двух-указательный подход не работает (отрицательные числа нарушают монотонность). Используем префиксные суммы и хэш-таблицу.

Заметим: $\text{sum}[l, r] = K \Leftrightarrow P[r+1] - P[l] = K \Leftrightarrow P[l] = P[r+1] - K$.

Алгоритм:
1. $\text{first}[0] = -1$ (пустой префикс «встретился» на позиции $-1$).
2. Для $r$ от 0 до $n-1$: обновляем $P_{cur} \mathrel{+}= A[r]$. Ищем $P_{cur} - K$ в $\text{first}$. Если нашли — обновляем ответ: $r - \text{first}[P_{cur} - K]$. Если $P_{cur}$ не в $\text{first}$ — записываем $\text{first}[P_{cur}] = r$.

Записываем $\text{first}$, а не счётчик — нам нужна наибольшая длина, то есть наиболее ранняя позиция.

Сложность: $O(n)$ среднее.

### Ответ

Длина наибольшего подотрезка с суммой $K$ находится за $O(n)$ с помощью хэш-таблицы первых вхождений префиксных сумм.

## Задача 5. Количество подотрезков с суммой, кратной p

**Условие.** Дан массив из $n$ целых чисел $A$ и простое $p$. Найдите количество пар $(l, r)$, $l \le r$, таких что $A[l] + \ldots + A[r]$ делится на $p$.

### Решение

$\text{sum}[l, r] \equiv 0 \pmod{p} \Leftrightarrow P[r+1] \equiv P[l] \pmod{p}$.

Считаем $P[i] \bmod p$. Если некоторое значение остатка встречается $c$ раз среди $P[0], P[1], \ldots, P[n]$, то оно даёт $\binom{c}{2}$ пар. Обновление массива частот:

1. Заводим массив $\text{cnt}[0..p-1] = 0$; $\text{cnt}[0] = 1$ (нулевой префикс).
2. Ведём $\text{psum} = 0$, $\text{ans} = 0$.
3. Для каждого $A[i]$: $\text{psum} = (\text{psum} + A[i] \bmod p + p) \bmod p$; $\text{ans} \mathrel{+}= \text{cnt}[\text{psum}]$; $\text{cnt}[\text{psum}]$++.

Сложность: $O(n + p)$ времени, $O(p)$ памяти.

### Ответ

Количество пар = $\sum_{r=0}^{p-1} \binom{\text{cnt}[r]}{2}$, считается за $O(n)$.

## Задача 6. Минимум в каждом окне длины k

**Условие.** Дан массив $A[0..n-1]$ и окно размера $k$. Для каждого $i$ от $0$ до $n - k$ найдите $\min(A[i], A[i+1], \ldots, A[i+k-1])$. Решите за $O(n)$.

### Решение

Применяем монотонную деку (дека с инвариантом: значения не убывают от начала к концу).

1. Создаём пустую деку $\text{dq}$ (хранит индексы).
2. Для $r$ от 0 до $n-1$:
   a. Пока дека непуста и $\text{dq.front()} \le r - k$: удаляем спереди.
   b. Пока дека непуста и $A[\text{dq.back()}] \ge A[r]$: удаляем сзади (они не смогут быть минимумом, пока $A[r]$ в окне).
   c. Добавляем $r$ в конец деки.
   d. Если $r \ge k - 1$: $\text{ans}[r - k + 1] = A[\text{dq.front()}]$.

Каждый элемент добавляется и удаляется из деки ровно по одному разу. Итого $O(n)$.

Трассировка для $A = [3, 1, 2, 5, 4]$, $k = 3$:

| r | Действие | Дека (индексы) | Минимум |
|---|---|---|---|
| 0 | push 0 | [0] | — |
| 1 | pop 0 (A[0]=3>=A[1]=1), push 1 | [1] | — |
| 2 | push 2 | [1, 2] | A[1]=1 |
| 3 | push 3 | [1, 2, 3]? нет, A[2]=2>=A[3]=5? нет. push 3 | A[1]=1 |
| 4 | pop 3 (A[3]=5>=A[4]=4), push 4; pop front? dq.front()=1, 1<=4-3=1, yes pop | [2, 4] | A[2]=2 |

Ответ: $[1, 1, 2]$.

### Ответ

Монотонная дека даёт $O(n)$. Для $A = [3, 1, 2, 5, 4]$, $k = 3$: минимумы $= [1, 1, 2]$.

## Задача 7. Прямоугольные запросы на матрице

**Условие.** Дана целочисленная матрица $A$ размера $n \times m$. Поступает $q$ запросов вида $(r_1, c_1, r_2, c_2)$: найти сумму элементов в прямоугольнике с левым верхним углом $(r_1, c_1)$ и правым нижним $(r_2, c_2)$ (0-indexed). Решите за $O(nm + q)$.

### Решение

1. Построение $P[n+1][m+1]$ за $O(nm)$:
$$P[i][j] = A[i-1][j-1] + P[i-1][j] + P[i][j-1] - P[i-1][j-1]$$

2. Ответ на запрос $(r_1, c_1, r_2, c_2)$ за $O(1)$:
$$\text{sum} = P[r_2+1][c_2+1] - P[r_1][c_2+1] - P[r_2+1][c_1] + P[r_1][c_1]$$

Обоснование формулы (принцип включения-исключения): $P[r_2+1][c_2+1]$ — сумма всего прямоугольника $[0,r_2] \times [0,c_2]$. Вычитаем полосу выше ($P[r_1][c_2+1]$) и левее ($P[r_2+1][c_1]$). Угол $[0,r_1) \times [0,c_1)$ вычтен дважды — прибавляем обратно $P[r_1][c_1]$.

### Ответ

Двумерный prefix строится за $O(nm)$, каждый запрос — $O(1)$. Общая сложность $O(nm + q)$.

## Задача 8. XOR-сегменты

**Условие.** Дан массив $A[0..n-1]$ из целых чисел. Ответьте на $q$ запросов: для каждого запроса $(l, r)$ найдите $A[l] \oplus A[l+1] \oplus \ldots \oplus A[r]$. Затем найдите количество подотрезков с нулевым XOR.

### Решение

**Часть 1.** Строим XOR-prefix: $P[0] = 0$, $P[i] = P[i-1] \oplus A[i-1]$.

$$\text{xor}[l, r] = P[r+1] \oplus P[l]$$

(XOR обратим: $x \oplus x = 0$, поэтому $P[r+1] \oplus P[l] = A[l] \oplus \ldots \oplus A[r]$.)

Каждый запрос — $O(1)$ после $O(n)$ предобработки.

**Часть 2.** $\text{xor}[l, r] = 0 \Leftrightarrow P[r+1] = P[l]$. Значит, нужно посчитать пары одинаковых значений в массиве $P[0..n]$. Если значение $v$ встречается $c_v$ раз, то оно даёт $\binom{c_v}{2}$ подотрезков. Используем хэш-таблицу частот: $O(n)$.

### Ответ

XOR на отрезке за $O(1)$. Количество подотрезков с нулевым XOR: $\sum_v \binom{c_v}{2}$, где $c_v$ — частота значения $v$ в $P[0..n]$.

## Задача 9. Два указателя на строках: наименьшее окно с k различными символами

**Условие.** Дана строка $s$ длины $n$ над алфавитом из латинских букв. Найдите длину наименьшего подотрезка (непрерывной подстроки), содержащего ровно $k$ различных символов. Если такого нет — выведите $-1$.

### Решение

Для фиксированного $l$ пусть $r^*(l)$ — наименьшее $r$ такое, что $[l, r]$ содержит ровно $k$ различных символов. При сдвиге $l$ вправо значение $r^*$ не уменьшается (монотонность: можно только «терять» символы, сужая окно).

Алгоритм (два указателя с частотным массивом):

1. $l = 0$, $r = -1$; $\text{freq}[26] = 0$; $\text{distinct} = 0$; $\text{ans} = \infty$.
2. Для $l$ от 0 до $n-1$:
   a. Пока $r + 1 < n$ и $\text{distinct} < k$: $r$++, обновляем $\text{freq}$ и $\text{distinct}$.
   b. Если $\text{distinct} = k$: $\text{ans} = \min(\text{ans}, r - l + 1)$.
   c. Убираем $s[l]$ из окна: уменьшаем $\text{freq}[s[l]]$; если стало 0 — $\text{distinct}$--.

Доказательство $O(n)$: $r$ движется только вправо, $l$ — тоже только вправо.

### Ответ

Метод двух указателей с частотным счётчиком за $O(n)$. Ответ — длина найденного окна или $-1$.

## Задача 10. Синтез: разделение массива на части с равными суммами

**Условие.** Дан массив из $n$ целых чисел (могут быть отрицательные). Найдите количество индексов $i$ ($1 \le i \le n-2$), таких что:
- сумма $A[0..i-1]$ равна сумме $A[i+1..n-1]$,
- и оба этих значения равны $\frac{S - A[i]}{2}$, где $S = A[0] + \ldots + A[n-1]$.

Иными словами, найдите «оси симметрии» массива по сумме (не считая само $A[i]$).

### Решение

1. Вычисляем $S = \sum A[i]$ за $O(n)$ (или через $P[n]$).
2. Строим массив префиксных сумм $P[0..n]$ за $O(n)$.
3. Для каждого индекса $i$ ($1 \le i \le n-2$):
   - Сумма левой части: $L = P[i]$.
   - Сумма правой части: $R = P[n] - P[i+1] = S - P[i+1]$.
   - Условие: $L = R$.

Это эквивалентно $P[i] = S - P[i+1]$, то есть $P[i] + P[i+1] = S$.

Поскольку $P[i+1] = P[i] + A[i]$, условие принимает вид:

$$2 P[i] + A[i] = S \quad \Leftrightarrow \quad 2 P[i] = S - A[i]$$

Для каждого $i$ от 1 до $n-2$ проверяем это за $O(1)$. Итого $O(n)$.

Дополнение: если нужно просто найти первый такой индекс, можно обойтись без построения полного массива $P$ — достаточно поддерживать скользящую сумму $L = P[i]$.

### Ответ

Индекс $i$ является осью симметрии тогда и только тогда, когда $2 P[i] = S - A[i]$. Алгоритм — $O(n)$ с помощью префиксных сумм.
