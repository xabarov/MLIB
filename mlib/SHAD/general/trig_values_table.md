# Табличные значения углов

Стандартные значения тригонометрических функций для часто используемых углов.

> Обозначения: `tan = tg`, `ctg = cot = 1/tan`.

| 135° | 3π/4      | √2/2      | -√2/2     | -1        | -1 |
| 150° | 5π/6      | 1/2       | -√3/2     | -√3/3     | -√3 |
| 180° | π         | 0         | -1        | 0         | не определен |
| 210° | 7π/6      | -1/2      | -√3/2     | √3/3      | √3 |
