## Задача 1

Узел решающего дерева содержит 40 объектов класса A и 10 объектов класса B. Вычислите критерий Джини для этого узла. Затем вычислите, чему равен критерий Джини для полностью чистого узла (все объекты одного класса) и для максимально загрязнённого (50 объектов A и 50 объектов B).

### Решение

Шаг 1. Доли классов:

$$p_A = \frac{40}{50} = 0.8, \quad p_B = \frac{10}{50} = 0.2$$

Шаг 2. Критерий Джини:

$$\text{Gini} = 1 - (p_A^2 + p_B^2) = 1 - (0.8^2 + 0.2^2) = 1 - (0.64 + 0.04) = 1 - 0.68 = 0.32$$

Шаг 3. Чистый узел (все A): $p_A = 1, p_B = 0$:

$$\text{Gini} = 1 - (1^2 + 0^2) = 0$$

Шаг 4. Максимально загрязнённый ($p_A = p_B = 0.5$):

$$\text{Gini} = 1 - (0.5^2 + 0.5^2) = 1 - 0.5 = 0.5$$

### Ответ

Gini для узла (40A, 10B) = **0.32**. Чистый узел: **0**. Максимальное загрязнение при равных долях: **0.5**.

## Задача 2

Даны три наблюдения: $(x_1, y_1) = (1, 3)$, $(x_2, y_2) = (2, 5)$, $(x_3, y_3) = (3, 8)$. Найдите коэффициенты линейной регрессии $y = wx + b$ методом нормального уравнения $w^* = (X^\top X)^{-1} X^\top y$.

### Решение

Шаг 1. Составим расширенную матрицу $X_b$ (первый столбец — единицы для $b$):

$$X_b = \begin{pmatrix} 1 & 1 \\ 1 & 2 \\ 1 & 3 \end{pmatrix}, \quad y = \begin{pmatrix} 3 \\ 5 \\ 8 \end{pmatrix}$$

Шаг 2. Вычислим $X_b^\top X_b$:

$$X_b^\top X_b = \begin{pmatrix} 1+1+1 & 1+2+3 \\ 1+2+3 & 1+4+9 \end{pmatrix} = \begin{pmatrix} 3 & 6 \\ 6 & 14 \end{pmatrix}$$

Шаг 3. Вычислим $X_b^\top y$:

$$X_b^\top y = \begin{pmatrix} 3+5+8 \\ 3+10+24 \end{pmatrix} = \begin{pmatrix} 16 \\ 37 \end{pmatrix}$$

Шаг 4. Найдём обратную матрицу ($\det = 3 \cdot 14 - 6 \cdot 6 = 42 - 36 = 6$):

$$(X_b^\top X_b)^{-1} = \frac{1}{6} \begin{pmatrix} 14 & -6 \\ -6 & 3 \end{pmatrix}$$

Шаг 5. Находим $w^* = (X_b^\top X_b)^{-1} X_b^\top y$:

$$w^* = \frac{1}{6} \begin{pmatrix} 14 & -6 \\ -6 & 3 \end{pmatrix} \begin{pmatrix} 16 \\ 37 \end{pmatrix} = \frac{1}{6} \begin{pmatrix} 224 - 222 \\ -96 + 111 \end{pmatrix} = \frac{1}{6} \begin{pmatrix} 2 \\ 15 \end{pmatrix} = \begin{pmatrix} 1/3 \\ 5/2 \end{pmatrix}$$

### Ответ

$b = 1/3 \approx 0.333$, $w = 5/2 = 2.5$. Модель: $\hat{y} = 2.5x + 0.333$.

## Задача 3

Вычислите вероятность $P(y=1 \mid x)$ для логистической регрессии с весами $w = (1.2, -0.8)$ и $b = 0.5$ для объекта $x = (2, 3)$. Затем вычислите бинарную кросс-энтропию (log-loss), если истинный класс $y = 1$.

### Решение

Шаг 1. Линейная комбинация:

$$z = w^\top x + b = 1.2 \cdot 2 + (-0.8) \cdot 3 + 0.5 = 2.4 - 2.4 + 0.5 = 0.5$$

Шаг 2. Sigmoid:

$$\sigma(0.5) = \frac{1}{1 + e^{-0.5}} = \frac{1}{1 + 0.6065} = \frac{1}{1.6065} \approx 0.6225$$

Шаг 3. Log-loss при $y = 1$ (один объект):

$$\mathcal{L} = -\log(\hat{p}) = -\log(0.6225) \approx -(-0.4745) = 0.4745$$

Для сравнения: если $\hat{p} = 0.9$, то $\mathcal{L} = -\log(0.9) \approx 0.105$ — потеря меньше.

### Ответ

$P(y=1 \mid x) \approx 0.622$. Log-loss $\approx 0.475$.

## Задача 4

Узел содержит 100 объектов: 60 класса A и 40 класса B. Рассматриваются два разбиения:

- **Разбиение 1:** левый узел (50A, 10B), правый узел (10A, 30B)
- **Разбиение 2:** левый узел (55A, 20B), правый узел (5A, 20B)

Вычислите взвешенный критерий Джини для каждого разбиения. Какое разбиение лучше?

### Решение

**Gini родительского узла:** $p_A = 0.6$, $p_B = 0.4$:
$\text{Gini} = 1 - (0.36 + 0.16) = 0.48$

**Разбиение 1:**

- Левый (60 объектов): $p_A = 50/60$, $p_B = 10/60$
  $$\text{Gini}_L = 1 - \left(\frac{50}{60}\right)^2 - \left(\frac{10}{60}\right)^2 = 1 - \frac{2500+100}{3600} = 1 - \frac{2600}{3600} \approx 0.278$$
- Правый (40 объектов): $p_A = 10/40 = 0.25$, $p_B = 30/40 = 0.75$
  $$\text{Gini}_R = 1 - (0.0625 + 0.5625) = 0.375$$
- Взвешенный: $\frac{60}{100} \cdot 0.278 + \frac{40}{100} \cdot 0.375 = 0.167 + 0.150 = 0.317$
- Прирост: $\Delta_1 = 0.48 - 0.317 = 0.163$

**Разбиение 2:**

- Левый (75 объектов): $p_A = 55/75$, $p_B = 20/75$
  $$\text{Gini}_L = 1 - \left(\frac{55}{75}\right)^2 - \left(\frac{20}{75}\right)^2 = 1 - \frac{3025+400}{5625} \approx 1 - 0.609 = 0.391$$
- Правый (25 объектов): $p_A = 5/25 = 0.2$, $p_B = 20/25 = 0.8$
  $$\text{Gini}_R = 1 - (0.04 + 0.64) = 0.32$$
- Взвешенный: $\frac{75}{100} \cdot 0.391 + \frac{25}{100} \cdot 0.32 = 0.293 + 0.080 = 0.373$
- Прирост: $\Delta_2 = 0.48 - 0.373 = 0.107$

### Ответ

**Разбиение 1 лучше** (прирост Gini = 0.163 против 0.107): оно создаёт более чистые дочерние узлы.

## Задача 5

Даны одномерные данные: класс $-1$: точки $\{1, 2\}$; класс $+1$: точки $\{4, 5\}$. Найдите оптимальную разделяющую гиперплоскость SVM (в 1D — это точка), опорные векторы и ширину зазора (margin).

### Решение

Шаг 1. SVM ищет гиперплоскость $w^\top x + b = 0$ с условиями:
- Для класса $-1$: $w \cdot x + b \leq -1$
- Для класса $+1$: $w \cdot x + b \geq +1$

Шаг 2. Опорные векторы — крайние точки каждого класса:
- $x^- = 2$ (класс $-1$): самая близкая к границе
- $x^+ = 4$ (класс $+1$): самая близкая к границе

Шаг 3. По симметрии граница находится в середине:
$$x^* = \frac{2 + 4}{2} = 3$$

Шаг 4. Ширина зазора: расстояние между опорными векторами:
$$\text{margin} = x^+ - x^- = 4 - 2 = 2$$

Шаг 5. Уравнение: при $w = 1$ и $b = -3$: $w \cdot 2 + b = -1$ ✓, $w \cdot 4 + b = 1$ ✓.

Геометрический зазор: $\frac{2}{\|w\|} = \frac{2}{1} = 2$.

### Ответ

Граница: $x = 3$. Опорные векторы: $x = 2$ и $x = 4$. Margin = **2**.

## Задача 6

Модель выдаёт логиты для трёх классов: $z = (2.0, 1.0, 0.5)$. Истинный класс — 1 (первый). Вычислите:
1. Вероятности softmax для каждого класса.
2. Кросс-энтропийную потерю.

### Решение

Шаг 1. Softmax:

$$P(k \mid x) = \frac{e^{z_k}}{\sum_{j} e^{z_j}}$$

Вычислим $e^{z_k}$:
$$e^{2.0} \approx 7.389, \quad e^{1.0} \approx 2.718, \quad e^{0.5} \approx 1.649$$

Сумма: $S = 7.389 + 2.718 + 1.649 = 11.756$

Вероятности:
$$P_1 = 7.389 / 11.756 \approx 0.628, \quad P_2 = 2.718 / 11.756 \approx 0.231, \quad P_3 = 1.649 / 11.756 \approx 0.140$$

Шаг 2. Кросс-энтропия (истинный класс — первый, $y_1 = 1$):

$$\mathcal{L} = -\log P_1 = -\log(0.628) \approx 0.465$$

Шаг 3. Проверка: $0.628 + 0.231 + 0.140 = 0.999 \approx 1$ ✓.

### Ответ

$P = (0.628, 0.231, 0.140)$. Потеря кросс-энтропии $\approx 0.465$.

## Задача 7

Обучается линейная регрессия методом градиентного спуска. На текущем шаге: $w = 2.0$, $b = 1.0$, learning rate $\eta = 0.1$. Один обучающий пример: $x = 3$, $y = 10$. Вычислите один шаг обновления весов по MSE ($\mathcal{L} = (y - \hat{y})^2$).

### Решение

Шаг 1. Текущее предсказание:
$$\hat{y} = w \cdot x + b = 2.0 \cdot 3 + 1.0 = 7.0$$

Шаг 2. Ошибка: $e = y - \hat{y} = 10 - 7 = 3$.

Шаг 3. Потеря: $\mathcal{L} = e^2 = 9$.

Шаг 4. Градиенты по MSE $= (y - \hat{y})^2$:

$$\frac{\partial \mathcal{L}}{\partial w} = -2 e \cdot x = -2 \cdot 3 \cdot 3 = -18$$

$$\frac{\partial \mathcal{L}}{\partial b} = -2 e = -2 \cdot 3 = -6$$

Шаг 5. Обновление:

$$w_{\text{new}} = w - \eta \cdot \frac{\partial \mathcal{L}}{\partial w} = 2.0 - 0.1 \cdot (-18) = 2.0 + 1.8 = 3.8$$

$$b_{\text{new}} = b - \eta \cdot \frac{\partial \mathcal{L}}{\partial b} = 1.0 - 0.1 \cdot (-6) = 1.0 + 0.6 = 1.6$$

Шаг 6. Проверка нового предсказания: $\hat{y} = 3.8 \cdot 3 + 1.6 = 11.4 + 1.6 = 13.0$. Ошибка стала $|10 - 13.0| = 3.0$... Шаг слишком большой — потеря выросла. При $\eta = 0.01$ было бы лучше.

### Ответ

После одного шага: $w = 3.8$, $b = 1.6$. Замечание: большой learning rate ($\eta = 0.1$) привёл к перешагу.

## Задача 8

Случайный лес обучается на 5 объектах: индексы $\{1, 2, 3, 4, 5\}$. Bootstrap-выборка для одного дерева: $\{2, 2, 4, 1, 3\}$. Определите, какие объекты out-of-bag (OOB) для этого дерева. При $n \to \infty$, какова ожидаемая доля OOB объектов?

### Решение

Шаг 1. Bootstrap-выборка содержит объекты: $\{1, 2, 3, 4\}$ (уникальные).

OOB-объекты (не вошли ни разу): **$\{5\}$**.

Шаг 2. Вероятность, что конкретный объект **не** попадёт в bootstrap-выборку размера $n$ при одном извлечении: $\frac{n-1}{n}$.

За $n$ независимых извлечений — вероятность не попасть ни разу:

$$P(\text{не попасть}) = \left(\frac{n-1}{n}\right)^n = \left(1 - \frac{1}{n}\right)^n$$

Шаг 3. При $n \to \infty$:

$$\lim_{n \to \infty} \left(1 - \frac{1}{n}\right)^n = e^{-1} \approx 0.368$$

Т.е. примерно **36.8%** объектов оказываются out-of-bag для каждого дерева.

### Ответ

OOB для данного дерева: объект $\{5\}$. При $n \to \infty$ доля OOB объектов $\to e^{-1} \approx 36.8\%$.

## Задача 9

Gradient Boosting с MSE. Обучающие данные: $y = (3, -1, 2, 4)$. Начальное приближение $F_0(x) = \bar{y}$ (среднее). Вычислите псевдо-остатки $r_i^{(1)}$ для первой итерации. Если следующее дерево $h_1$ предсказывает эти остатки как константу в одном листе (полное дерево), найдите $F_1(x)$ при $\eta = 0.5$.

### Решение

Шаг 1. Начальное приближение:
$$F_0 = \bar{y} = \frac{3 + (-1) + 2 + 4}{4} = \frac{8}{4} = 2$$

Шаг 2. Псевдо-остатки (для MSE — обычные остатки):

$$r_i^{(1)} = y_i - F_0 = \begin{cases} 3 - 2 = +1 \\ -1 - 2 = -3 \\ 2 - 2 = 0 \\ 4 - 2 = +2 \end{cases}$$

Шаг 3. Дерево $h_1$ обучается на $(x_i, r_i^{(1)})$. Если оно — константная модель (один лист), то предсказывает среднее остатков:

$$h_1 = \bar{r} = \frac{1 + (-3) + 0 + 2}{4} = \frac{0}{4} = 0$$

В этом случае GBM не улучшается. Для реального улучшения нужно дерево с разбиениями. Допустим, дерево разбивает на два листа:
- Лист 1 (объекты 1 и 4): $h_1^{(A)} = (1 + 2)/2 = 1.5$
- Лист 2 (объекты 2 и 3): $h_1^{(B)} = (-3 + 0)/2 = -1.5$

Шаг 4. Обновление при $\eta = 0.5$:

$$F_1(x) = F_0 + \eta \cdot h_1(x)$$

- Объект 1: $F_1 = 2 + 0.5 \cdot 1.5 = 2.75$
- Объект 2: $F_1 = 2 + 0.5 \cdot (-1.5) = 1.25$
- Объект 3: $F_1 = 2 + 0.5 \cdot (-1.5) = 1.25$
- Объект 4: $F_1 = 2 + 0.5 \cdot 1.5 = 2.75$

Новые остатки: $(0.25, -2.25, 0.75, 1.25)$ — стали меньше по абсолютной величине.

### Ответ

Псевдо-остатки: $r^{(1)} = (1, -3, 0, 2)$. После одной итерации GBM с $\eta = 0.5$: $F_1 = (2.75, 1.25, 1.25, 2.75)$.

## Задача 10 (Синтез)

Вам нужно предсказать вероятность оттока клиента банка. Обучающая выборка: 50 000 объектов, 80 признаков (числовые, категориальные, 15% пропусков), сильный дисбаланс классов (95% не уходят, 5% уходят). Заказчик требует объяснять решения по конкретному клиенту. Какую модель выбрать? Опишите полный пайплайн и ключевые решения.

### Решение

**Анализ условий:**

| Аспект | Вывод |
|---|---|
| 50 000 объектов, 80 признаков | Достаточно для GBM/RF, не нужны линейные только из-за скорости |
| Категориальные признаки | CatBoost или OneHotEncoding+RF |
| 15% пропусков | GBM/RF справляются нативно |
| Дисбаланс 95/5% | Нужна стратегия: `class_weight`, SMOTE или threshold tuning |
| Объяснимость | SHAP-значения (любая модель) или Лог. регрессия (intrinsically interpretable) |

**Рекомендуемый подход:** Gradient Boosting (CatBoost/XGBoost) + SHAP.

**Пайплайн:**

```python
import pandas as pd
from sklearn.model_selection import StratifiedKFold, cross_val_score
from sklearn.metrics import roc_auc_score
# pip install catboost shap
from catboost import CatBoostClassifier
import shap

df = pd.read_csv("bank_churn.csv")
cat_cols = df.select_dtypes("object").columns.tolist()
X = df.drop("churn", axis=1)
y = df["churn"]

# CatBoost нативно обрабатывает категориальные и пропуски
model = CatBoostClassifier(
    iterations=500,
    learning_rate=0.05,
    depth=6,
    cat_features=cat_cols,
    auto_class_weights="Balanced",  # дисбаланс классов
    early_stopping_rounds=50,
    eval_metric="AUC",
    random_seed=42,
    verbose=False,
)

# Стратифицированная CV (сохраняет долю классов)
cv = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(model, X, y, cv=cv, scoring="roc_auc")
print(f"ROC-AUC: {scores.mean():.4f} ± {scores.std():.4f}")

# Объяснимость через SHAP
model.fit(X, y)
explainer = shap.TreeExplainer(model)
shap_values = explainer.shap_values(X.iloc[:100])
shap.summary_plot(shap_values, X.iloc[:100])
# Для конкретного клиента:
shap.force_plot(explainer.expected_value, shap_values[0], X.iloc[0])
```

**Ключевые решения:**

1. **Метрика:** ROC-AUC (не Accuracy, т.к. дисбаланс 95/5%).
2. **Дисбаланс:** `auto_class_weights="Balanced"` (CatBoost) или `scale_pos_weight` (XGBoost).
3. **Объяснимость:** SHAP даёт глобальное (summary_plot) и локальное (force_plot) объяснение.
4. **Пороговая настройка:** по умолчанию порог 0.5 не оптимален при дисбалансе — выбрать по precision-recall кривой.

### Ответ

**Выбор:** CatBoost (нативные категориальные и пропуски) + SHAP для объяснений + ROC-AUC как метрика + Balanced class weights для дисбаланса. Это даёт одновременно высокую точность и требуемую объяснимость.
