# Рак: заявление об accuracy 99.99%

**Вопрос (EN)**: Team built a system predicting cancer from X-rays with 99.99% accuracy. How would you respond?

## Ответ

### Критический взгляд

1. **Class imbalance**  
   Рак — редкий класс (доля порядка 0.01–1%). Если модель всегда предсказывает «нет рака», accuracy будет ~99%+. Высокая accuracy при дисбалансе ничего не говорит о качестве.

2. **Метрики**  
   Важны recall (sensitivity), precision, F1, AUC-PR. Ложноотрицательные в медицине опасны — пропущенный рак.

3. **Data leakage**  
   Проверить, не попали ли целевые метки или производные в признаки; правильность split (нет утечки из test в train).

4. **Размер и представительность выборки**  
   Достаточно ли данных, покрывают ли они разные типы опухолей, аппараты, популяции.

5. **Практика**  
   Сравнение с экспертами, анализ на подгруппах, временная валидация.

### Резюме

Высокая accuracy при дисбалансе подозрительна. Нужны recall, precision, F1, ROC-AUC, анализ ошибок и проверка на утечку данных.
