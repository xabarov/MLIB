# MPE vs MAP

**Вопрос (EN)**: How do MPE and MAP differ? Give an example when they would produce different results.

## Ответ

### MPE (Most Probable Explanation)

\[
\text{MPE} = \arg\max_x P(X=x | E=e)
\]

Максимизация совместной вероятности всех переменных при наблюдаемых \(E\). Ищет наиболее вероятную полную конфигурацию.

### MAP (Maximum A Posteriori)

\[
\text{MAP} = \arg\max_x P(X_q=x | E=e)
\]

Максимизация апостериорной вероятности только по запрашиваемым переменным \(X_q\), маргинализуя по остальным.

### Различие

- MPE: полная конфигурация всех переменных.
- MAP: максимум по подмножеству переменных (маргинализация по прочим).

### Пример, когда MPE и MAP дают разные результаты

Пусть \(X, Y \in \{0,1\}\). При наблюдении \(E\): P(1,1)=0.4, P(0,1)=0.35, P(0,0)=0.2, P(1,0)=0.05.

**MPE:** \(\arg\max_{x,y} P(X,Y|E)\) = (1,1) — наибольшая совместная вероятность.

**MAP по X:** \(P(X=0|E)=0.35+0.2=0.55\), \(P(X=1|E)=0.45\). MAP даёт \(X=0\).

Итог: MPE возвращает (1,1), т.е. X=1; MAP по X возвращает X=0. Результаты различаются.
