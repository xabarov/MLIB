# Сходимость алгоритма

**Вопрос (EN)**: When we say an algorithm converges, what does convergence mean? How do we know when a model has converged?

## Ответ

### Что такое сходимость

Алгоритм сходится, когда последовательность значений функции потерь (или параметров) стабилизируется: дальнейшие итерации почти не меняют результат.

### Когда считать, что модель сошлась

1. **Плато loss** — \(|L_{t} - L_{t-1}| < \epsilon\) или \(\frac{|L_t - L_{t-1}|}{|L_{t-1}|} < \epsilon\) за несколько итераций; loss выходит на ровное плато.

2. **Малое изменение весов** — \(\|w_t - w_{t-1}\| < \epsilon\); иногда используют вместо/в дополнение к loss при SGD с шумом.

3. **Градиент близок к нулю** — \(\|\nabla L(w)\| < \epsilon\); часто не проверяют каждый шаг из-за вычислительной стоимости.

4. **Заданное число эпох** — обучение останавливается после N эпох.

5. **Early stopping** — loss на validation перестаёт уменьшаться или начинает расти; сохраняют модель до ухудшения — частый практический критерий сходимости.

6. **Практическая стабильность** — метрика на validation почти не меняется.
