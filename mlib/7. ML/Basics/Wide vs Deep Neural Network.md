# Широкая vs глубокая нейросеть при одинаковом числе параметров

**Вопрос (EN)**: If we have a wide NN and a deep NN with the same number of parameters, which one is more expressive and why?

## Ответ

**Глубокая сеть обычно более выразительна.**

### Причины

1. **Композиция функций**  
   Глубокая сеть реализует композицию нелинейных преобразований: \(f_1(f_2(...(f_n(x))))\). Такие иерархические представления могут выражать сложные зависимости (например, иерархию признаков в изображениях).

2. **Экспоненциальный рост выразительности**  
   Теоретически глубина может давать экспоненциальный прирост выразительности при том же числе параметров (работа в духе моногенных функций, экспоненциальные разделяющие границы и т.п.).

3. **Широкая однослойная сеть**  
   По сути — один слой нелинейных преобразований. Она может приближать любую непрерывную функцию (теорема об универсальной аппроксимации), но для сложных зависимостей потребуется гораздо больше параметров, чем при разбиении на несколько слоёв.

4. **Практический опыт**  
   В CV и NLP глубокие архитектуры (ResNet, Transformer) дают лучшее качество при сопоставимом числе параметров.

### Итог

При фиксированном бюджете параметров глубина обычно даёт более выразительную модель, чем ширина.
