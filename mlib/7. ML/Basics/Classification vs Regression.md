# Классификация и регрессия

**Вопрос (EN)**: What makes a classification problem different from a regression problem? Can a classification problem be turned into a regression problem and vice versa?

## Ответ

### В чём разница

- **Классификация** — предсказание дискретной метки (класса). Примеры: спам/не спам, тип объекта на изображении. Метрики: accuracy, F1, precision, recall.
- **Регрессия** — предсказание непрерывной величины. Примеры: цена, температура, время. Метрики: MSE, MAE, RMSE.

### Классификация → регрессия

Можно, если предсказывать непрерывную вероятность класса (0–1) или оценку «уверенности» вместо дискретной метки. Логистическая регрессия как раз предсказывает \(P(y=1|x)\). Дополнительно можно регрессировать расстояние до границы решения (margin).

### Регрессия → классификация

Можно, если разбить целевую переменную на интервалы (bins) и решать задачу классификации по этим интервалам. Пример: вместо точной цены предсказывать диапазон (дешёвый/средний/дорогой). Можно также использовать регрессию для learning-to-rank, а потом порог для получения классов.
