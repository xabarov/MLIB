# Параметрические и непараметрические методы

**Вопрос (EN)**: What's the difference between parametric methods and non-parametric methods? Give an example of each. When should we use one and when the other?

## Ответ

### Параметрические методы

- Имеют фиксированное число параметров, не зависящее от размера выборки.
- Строят предположение о виде распределения (например, нормальность) или структуре модели.
- Примеры: линейная регрессия, логистическая регрессия, наивный Байес.

### Непараметрические методы

- Число «параметров» растёт с ростом данных (например, число точек, по которым строят оценки).
- Не фиксируют строгую форму модели заранее.
- Примеры: k-NN, деревья решений, ядерная регрессия, метод опорных векторов с RBF.

### Когда что использовать

**Параметрические:**
- Данных мало.
- Есть обоснованные предположения о форме зависимости.
- Нужна быстрая инференс-модель и интерпретируемость.
- Ограничения по памяти и вычислениям.

**Непараметрические:**
- Данных много.
- Форма зависимости неизвестна или сложная.
- Приоритет — качество, а не скорость и интерпретируемость.
- Достаточно ресурсов для хранения и вычислений.
