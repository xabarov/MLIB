# Объяснение типов обучения: supervised, unsupervised, weakly supervised, semi-supervised и active learning

**Вопрос (EN)**: Explain supervised, unsupervised, weakly supervised, semi-supervised, and active learning.

## Ответ

### Supervised Learning (обучение с учителем)
Модель обучается на размеченных данных: каждой выборке соответствует целевая метка (класс или число). Задача — научиться предсказывать метку для новых объектов. Примеры: классификация (спам/не спам), регрессия (прогноз цены).

### Unsupervised Learning (обучение без учителя)
Модель получает только признаки без меток. Задача — находить структуру в данных: кластеры, аномалии, латентные представления. Примеры: k-means, PCA, автоэнкодеры.

### Weakly Supervised Learning (слабое обучение с учителем)
Метки неполные, неточные или косвенные: например, только наличие класса, но не его расположение (image-level labels для object detection), или шумные метки. Модель должна учиться при неполной или неточной информации.

### Semi-Supervised Learning (полуавтоматическое обучение)
Есть небольшое число размеченных и много неразмеченных данных. Алгоритмы используют структуру неразмеченных данных (кластеры, непрерывность) для улучшения предсказаний. Методы: self-training, pseudo-labeling, consistency regularization.

### Active Learning (активное обучение)
Модель сама выбирает, какие примеры просить разметить эксперту. Цель — максимизировать качество при минимизации числа запросов. Критерии отбора: uncertainty sampling, query-by-committee, expected model change и др.
