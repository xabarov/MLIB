# Markov Chain Monte Carlo (MCMC)

**Вопрос (EN)**: Explain Markov chain Monte Carlo sampling.

## Ответ

**MCMC** — семейство методов для выборки из сложных многомерных распределений, когда прямое сэмплирование затруднено.

### Идея

Строится марковская цепь, чьё стационарное распределение совпадает с целевым \(p(x)\). Сэмплирование выполняется по шагам цепи; после «прогрева» (burn-in) точки цепи приближённо подчиняются \(p(x)\).

### Основные алгоритмы

1. **Metropolis–Hastings**  
   Предложение новой точки \(x'\), приём с вероятностью  
   \(A = \min\left(1, \frac{p(x')q(x|x')}{p(x)q(x'|x)}\right)\).  
   Достаточно знать \(p\) с точностью до нормировочной константы.

2. **Gibbs sampling**  
   Обновление по одной переменной за раз при условии остальных. Удобно, когда условные распределения \(p(x_i | x_{-i})\) легко сэмплировать.

### Применения в ML

- Байесовский вывод (posterior sampling).
- Обучение ограниченных машин Больцмана (RBM).
- Вариационный вывод и сэмплирование из сложных совместных распределений.
