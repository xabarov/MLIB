# Сэмплирование из высокоразмерных данных

**Вопрос (EN)**: If you need to sample from high-dimensional data, which sampling method would you choose?

## Ответ

В высокой размерности наивные методы (например, rejection sampling) становятся неэффективны из‑за «проклятия размерности».

### Подходящие методы

1. **MCMC (Markov Chain Monte Carlo)**  
   Metropolis-Hastings, Gibbs sampling — строят цепь с нужным стационарным распределением, не требуют знать нормировку.

2. **Вариационные методы**  
   Оценка распределения параметрической семьёй (например, гауссовой) и оптимизация по параметрам. Масштабируется лучше MCMC.

3. **Langevin dynamics / MALA**  
   Использование градиентов для предложений в MCMC; лучше подходит для непрерывных распределений.

4. **Importance sampling**  
   При удачном proposal может давать хорошие оценки; сложность — выбор proposal в высокой размерности.

5. **Сэмплирование в латентном пространстве**  
   Сначала снизить размерность (PCA, VAE), затем сэмплировать в низкоразмерном пространстве и отображать обратно.

### Чего избегать

- Rejection sampling — очень низкая acceptance rate в высокой размерности.
- Наивная grid sampling — экспоненциальный рост числа точек.
