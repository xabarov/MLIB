# Предположения линейной регрессии

**Вопрос (EN)**: What are the basic assumptions for linear regression?

## Ответ

1. **Линейность** — \(E[Y|X] = X\beta\); связь линейна по параметрам. *Нарушение:* смещённые оценки; нелинейность можно моделировать (polynomial, log, и т.д.).

2. **Гомоскедастичность** — дисперсия ошибок постоянна для всех X. *Нарушение:* стандартные ошибки и доверительные интервалы ненадёжны; использовать robust/sandwich SE или WLS.

3. **Независимость ошибок** — остатки не коррелированы. *Нарушение:* заниженные SE, завышенная значимость; серийная корреляция (временные ряды) — использовать HAC, clustered SE.

4. **Нормальность остатков** — для inference (t-тесты, F-тесты, доверительные интервалы). *Нарушение:* при больших n — асимптотически всё ещё ОК; при малых n — bootstrap или robust methods.

5. **Нет мультиколлинеарности** — признаки не являются линейными комбинациями друг друга. *Нарушение:* высокие SE, нестабильные оценки β; можно убрать/объединить признаки, регуляризация.

6. **Нет эндогенности** — ошибки не коррелируют с X (нет omitted variable bias). *Нарушение:* смещённость β; нужны IV, контроль дополнительных переменных, квази-эксперименты.
