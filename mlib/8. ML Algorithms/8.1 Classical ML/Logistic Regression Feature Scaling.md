# Feature scaling для логистической регрессии

**Вопрос (EN)**: What happens if we don't apply feature scaling to logistic regression?

## Ответ

### Без масштабирования

1. **Разная скорость сходимости** — признаки в диапазоне 0–1 vs 0–1000: разные масштабы градиентов, сходимость итерационных методов (SGD) замедляется.

2. **Численная нестабильность** — при вычислении exp большие веса могут приводить к overflow/underflow.

3. **Регуляризация** — L1/L2 штрафуют большие веса; unscaled признаки с большим разбросом получают малые веса, а малые признаки — большие, что искажает эффект регуляризации.

4. **Интерпретация** — веса перестают быть сопоставимыми между признаками.

### Рекомендация

StandardScaler (zero mean, unit variance) или MinMaxScaler перед обучением.
