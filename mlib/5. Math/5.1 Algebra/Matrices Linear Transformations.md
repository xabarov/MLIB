# Матрицы как линейные преобразования

**Вопрос (EN)**: Why are matrices linear transformations? Inverse, determinant?

## Ответ

### Линейное преобразование

Умножение \(y = Ax\) линейно: \(A(\alpha x + \beta z) = \alpha Ax + \beta Az\). Любое линейное отображение между конечномерными пространствами задаётся матрицей в выбранном базисе.

### Обратная матрица

\(A^{-1}\) — такая матрица, что \(AA^{-1} = A^{-1}A = I\). Обратимы только квадратные матрицы с ненулевым определителем. Обратная единственна, если существует.

### Определитель

Определитель — signed объём параллелепипеда, натянутого на строки (столбцы) матрицы. \(\det A = 0\) ⇔ строки/столбцы линейно зависимы ⇔ матрица необратима. При умножении строки на \(t\) определитель умножается на \(t\).

### След и определитель через собственные значения

\(\text{trace}(A) = \sum \lambda_i\), \(\det(A) = \prod \lambda_i\). Для матрицы 4×4 с λ = 3,3,2,-1: trace = 7, det = -18.
