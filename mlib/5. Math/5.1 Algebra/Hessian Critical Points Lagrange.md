# Гессиан, критические точки и метод Лагранжа

**Вопрос (EN)**: How use Hessian to test critical points? Given f(x,y)=4x²-y, constraint x²+y²=1. Find max and min.

## Ответ

### Классификация критических точек по гессиану

В стационарной точке \(\nabla f = 0\). Матрица Гессе \(H\) — матрица вторых производных.

- **H положительно определена** (все \(\lambda_i > 0\)) → локальный минимум
- **H отрицательно определена** (все \(\lambda_i < 0\)) → локальный максимум
- **H неопределена** (есть \(\lambda_i > 0\) и \(\lambda_j < 0\)) → седловая точка
- **H вырождена** (есть \(\lambda_i = 0\)) → нужен анализ старших производных

### Метод множителей Лагранжа

Функция \(f(x,y) = 4x^2 - y\), ограничение \(g(x,y) = x^2 + y^2 - 1 = 0\).

\(\nabla f = \lambda \nabla g\):
- \(8x = 2\lambda x\)
- \(-1 = 2\lambda y\)

Из первого: либо \(x=0\), либо \(\lambda = 4\).  
При \(x=0\): \(y^2=1\), \(y=\pm1\). Тогда \(-1=2\lambda y\) → \(\lambda = -1/(2y)\). При \(y=1\): \(\lambda = -1/2\); при \(y=-1\): \(\lambda = 1/2\).  
При \(\lambda=4\): \(-1=8y\) → \(y=-1/8\). Тогда \(x^2 = 1 - 1/64 = 63/64\), \(x = \pm\sqrt{63}/8\).

Значения:
- \(f(0,1) = -1\)
- \(f(0,-1) = 1\)
- \(f(\pm\sqrt{63}/8, -1/8) = 4 \cdot 63/64 + 1/8 = 63/16 + 1/8 = 65/16\)

**Минимум:** \(-1\) в \((0, 1)\). **Максимум:** \(65/16\) в \((\pm\sqrt{63}/8, -1/8)\).
