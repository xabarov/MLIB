# Линейные системы Ax=b и псевдообратная матрица

**Вопрос (EN)**: Given A∈R^{n×m}, b∈R^n. Find x such that Ax=b. When unique solution? Why multiple solutions when cols>rows? Matrix with no inverse — how to solve? What is pseudoinverse?

## Ответ

### Решение Ax = b

Система \(Ax = b\). Решение существует, если \(b \in \text{col}(A)\), т.е. rank(A) = rank([A|b]).

### Когда решение единственное

При **m = n** и **det(A) ≠ 0**: \(x = A^{-1}b\), решение единственно.

В общем случае: единственность при **полном столбцовом ранге** (rank(A) = m). Тогда \(A^TA\) обратима и \(x = (A^TA)^{-1}A^T b\).

### Почему много решений при cols > rows

Если \(m > n\) и rank(A) = n, то \(\text{null}(A)\) ненулевое: dim(null) = m − n. Любое частное решение \(x_p\) плюс произвольный вектор из null даёт другое решение: \(x = x_p + x_n\), \(Ax_n = 0\). Решений бесконечно много (аффинное подпространство размерности m−n).

### Псевдообратная матрица (Moore–Penrose)

При необратимой A используют **псевдообратную** \(A^+\):

- При **полном столбцовом ранге** (m ≤ n): \(A^+ = (A^TA)^{-1}A^T\). Минимизирует \(\|Ax - b\|^2\), даёт решение с минимальной нормой \(\|x\|\).
- При **полном строчном ранге** (n ≤ m): \(A^+ = A^T(AA^T)^{-1}\).
- В общем случае: через SVD \(A = U\Sigma V^T\), тогда \(A^+ = V\Sigma^+ U^T\), где \(\Sigma^+\) — диагональ с \(1/\sigma_i\) для \(\sigma_i > 0\) и 0 иначе.

**Наименьшие квадраты:** \(x = A^+ b\) даёт минимум \(\|Ax - b\|^2\); при неединственности — решение с минимальной нормой.
