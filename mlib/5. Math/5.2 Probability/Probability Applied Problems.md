## 9. Хранилище: 5 сбоев в год

**Вопрос:** Система падала 5 раз за год, сбои независимы. P(сбой в следующий месяц)? P(сбой в произвольный момент)?

**Ответ:** Модель Пуассона: λ = 5/12 сбоев в месяц. P(≥1 сбой в месяц) = 1 − e^{−λ} ≈ 1 − e^{−5/12}.  
«В произвольный момент» — предельная интенсивность: при очень малом Δt вероятность ≈ λ·Δt; формально требуется уточнение модели (например, Poisson process).

## 10. Классификатор: 10 ошибок из 100

**Вопрос:** 10 ошибок из 100. Все предсказания независимы. P(следующие 20 всех верны)?

**Ответ:** Оценка P(правильно) ≈ 90/100 = 0.9. При независимости P(20 подряд верных) = 0.9^20 ≈ 0.12.

## 13. Производители чипов A и B

**(a)** P(дефектный) = 0.5·0.3 + 0.5·0.7 = 0.5.  
**(b)** Байес: P(компания A | первый годен) ∝ P(годен|A)·P(A) = 0.7·0.5 = 0.35; P(компания B|первый годен) ∝ 0.3·0.5 = 0.25. Нормировка: P(A|годен) = 0.35/0.6 = 7/12, P(B|годен) = 5/12.  
P(второй годен) = P(годен|A)·P(A|годен) + P(годен|B)·P(B|годен) = 0.7·(7/12) + 0.3·(5/12) = 64/120 = 8/15.

## 14. Редкая болезнь, тест с 1% ошибками

**(a)** Prior P(болен) = 1/10000. P(положительный|болен)=0.99, P(положительный|здоров)=0.01.  
P(болен|положительный) = P(пол|болен)P(болен) / P(пол) = (0.99·0.0001) / (0.99·0.0001 + 0.01·0.9999) ≈ 0.0098 (около 1%).  

**(b)** Два независимых положительных теста. Posterior после первого теста становится prior для второго.  

После первого положительного: P(болен|пол₁) ≈ 0.0098 (из пункта a).  
Prior для второго: P(болен) = 0.0098, P(здоров) = 0.9902.  
P(пол₂|болен) = 0.99, P(пол₂|здоров) = 0.01.  

P(пол₂) = 0.99·0.0098 + 0.01·0.9902 ≈ 0.0097 + 0.0099 = 0.0196.  
P(болен|два пол) = P(пол₂|болен)·P(болен) / P(пол₂) = 0.99·0.0098 / 0.0196 ≈ **0.495 ≈ 49.5%**.

## 15. Знакомства: 10 прилагательных из 50, совпадение ≥5

**Вопрос:** Jack и Jin случайно выбирают по 10 из 50. «Матч» — не менее 5 общих. P(матч)?

**Ответ:** Гипергеометрическое распределение. P(k общих) = C(10,k)·C(40,10−k) / C(50,10).  
P(матч) = Σ_{k=5}^{10} P(k общих) ≈ **0.018** (≈ 1.8%). Основной вклад даёт k=5 (≈ 1.6%), остальные слагаемые малы.

## 16. Рост: смесь мужского и женского распределений

**Вопрос:** Пол неизвестен. Рост мужчин N(μ_m, σ_m²), женщин N(μ_f, σ_f²). PDF роста A.

**Ответ:** По формуле полной вероятности:  
f(h) = P(мужчина)·f(h|мужчина) + P(женщина)·f(h|женщина) = 0.5·N(h;μ_m,σ_m²) + 0.5·N(h;μ_f,σ_f²).

## 17. Погодные приложения: туман в SF

**Вопрос:** Три приложения, каждое ошибается с вероятностью 1/3. Все предсказывают туман. Prior P(туман)=0.5. P(туман|все три — туман)?

**Ответ:**  
- **Независимые ошибки:** P(все 3 туман|туман) = (2/3)³ = 8/27, P(все 3 туман|не туман) = (1/3)³ = 1/27.  
  P(все 3) = 0.5·(8/27) + 0.5·(1/27) = 9/54.  
  P(туман|все 3) = (0.5·8/27) / (9/54) = (4/27) / (9/54) = 8/9 ≈ **0.89**.  
- **Зависимые:** если приложения используют одни данные, ошибки коррелированы, обновление prior слабее.

## 18. German tank problem: оценка d

**Вопрос:** n выборок из U[0,d]. Оценить d.

**Ответ:** MLE: \(\hat{d} = \max_i x_i\). Но он смещён вниз. Несмещённая оценка: \(\hat{d} = \frac{n+1}{n} \max_i x_i\). Вариант: \(\hat{d} = \max + \frac{\max}{n} - 1\) для дискретного U{1,...,d}.

## 19. Ожидаемое число дней до X > 0.5

**Вопрос:** X ~ N(0,1), одна реализация в день. Сколько в среднем дней до первого X > 0.5?

**Ответ:** P(X>0.5) = 1−Φ(0.5) ≈ 0.31. Число попыток до первого успеха — геометрическое с p≈0.31. E[дней] = 1/p ≈ 3.2.

## 20. Размер класса и день рождения

**Вопрос:** Размер класса n. P(хотя бы у одного тот же день рождения, что у меня) > 50%?

**Ответ:** P(никто не совпал) = (364/365)^n. Нужно (364/365)^n < 0.5, т.е. n > ln(0.5)/ln(364/365) ≈ 253. Ответ: n ≥ 254.

## 21. Vegas: стратегия с гарантированным выигрышем?

**Вопрос:** p — вероятность выигрыша (удвоение ставки), 1−p — проигрыш. Неограниченный капитал. Есть ли стратегия с гарантированным плюсом при любом p?

**Ответ:** Нет. Martingale (удвоение ставки после проигрыша) даёт плюс с вероятностью 1 при неограниченном капитале, но ожидаемый выигрыш при p<0.5 отрицательный; при p≥0.5 и ограниченном числе игр — иной анализ. При любом фиксированном p≠1 «гарантированная» прибыль недостижима из-за ограничений математического ожидания.

## 22. Ожидаемое число подбрасываний до двух орлов подряд

**Вопрос:** Честная монета. Сколько в среднем подбрасываний до HH?

**Ответ:** Введём E₀ — мат. ожидание с нуля (ни одного H подряд), E₁ — после одной H.  

- Из E₀: с вероятностью 1/2 выпадает T → возврат в E₀; с вероятностью 1/2 — H → переход в E₁.  
  E₀ = 1 + (1/2)E₀ + (1/2)E₁.  

- Из E₁: с вероятностью 1/2 — H, конец (0 доп. бросков); с вероятностью 1/2 — T → возврат в E₀.  
  E₁ = 1 + (1/2)·0 + (1/2)E₀ = 1 + E₀/2.  

Подставляя E₁ в первое уравнение: E₀ = 1 + E₀/2 + (1 + E₀/2)/2 = 1.5 + 3E₀/4 ⇒ E₀/4 = 1.5 ⇒ **E₀ = 6**.

## 23. Law of small numbers: города с низким/высоким уровнем болезней

**Вопрос:** Топ-3 города по низкому и топ-3 по высокому уровню болезни почек — все с населением <5000. Почему?

**Ответ:** Закон малых чисел: при малой выборке дисперсия относительной частоты велика. В маленьких городах случайные флуктуации дают экстремальные доли. Большие города ближе к среднему по популяции.

## 24. MLE для экспоненциального распределения

**Вопрос:** Вывести MLE для Exp(λ).

**Ответ:** f(x|λ)=λe^{−λx}, x≥0. L(λ)=∏ λe^{−λx_i} = λ^n e^{−λ Σx_i}. ln L = n ln λ − λ Σx_i. d/dλ = n/λ − Σx_i = 0 → \(\hat{\lambda} = n / \sum x_i = 1/\bar{x}\).
