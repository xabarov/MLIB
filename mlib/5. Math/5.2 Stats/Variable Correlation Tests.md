# Тесты на связь переменных

**Вопрос (EN)**: What happens to regression if independent variables are strongly correlated? Test independence for categorical variables? For continuous?

## Ответ

### Сильная корреляция «независимых» переменных (мультиколлинеарность)

- Нестабильность оценок коэффициентов, большие стандартные ошибки.
- Знаки и величины коэффициентов могут быть искажены.
- Матрица X^TX почти вырождена.
- Решения: отбор признаков, регуляризация (Ridge), PCA, увеличение данных.

**VIF (Variance Inflation Factor):** для признака j
\[
\text{VIF}_j = \frac{1}{1 - R_j^2}
\]
где \(R_j^2\) — R² регрессии X_j на остальные признаки. VIF > 5–10 обычно указывает на серьёзную мультиколлинеарность; VIF > 10 — критично.

### Категориальные переменные

**Chi-squared test:** таблица сопряжённости, \(H_0\) — независимость. Статистика \(\chi^2 = \sum \frac{(O-E)^2}{E}\). При \(H_0\) асимптотически \(\chi^2\) с (r−1)(c−1) степенями свободы.

**Числовой пример (2×2):** Пол vs Любимый цвет (красный/синий). Наблюдаемые: M–красный 30, M–синий 20, F–красный 10, F–синий 40.  
Суммы: M=50, F=50, красный=40, синий=60, всего 100. Ожидаемые при независимости: E(M,красный)=50·40/100=20, E(M,синий)=30, E(F,красный)=20, E(F,синий)=30.  
\(\chi^2 = \frac{(30-20)^2}{20} + \frac{(20-30)^2}{30} + \frac{(10-20)^2}{20} + \frac{(40-30)^2}{30} = 5 + 3.33 + 5 + 3.33 \approx 16.67\).  
С 1 степенью свободы p ≈ 0.00005 — зависимость значима.

### Непрерывные переменные

- **Корреляция Пирсона** — линейная связь; H₀: ρ=0.
- **Спирмена, Кендалла** — ранговые корреляции, устойчивее к выбросам.
- **Distance correlation** — выявляет нелинейные связи, равна 0 только при независимости.
- Визуализация: scatter plot, коррелограммы.
