# Частотная и байесовская статистика

**Вопрос (EN)**: Explain frequentist vs Bayesian statistics.

## Ответ

### Частотный подход

- Вероятность — предел частоты при повторении эксперимента.
- Параметры — фиксированные, неизвестные константы.
- Доверительные интервалы: интервал либо содержит параметр, либо нет; 95% относится к процедуре, не к конкретному интервалу.
- p-value: вероятность данных при нулевой гипотезе.
- Оценки: MLE, метод моментов.

### Байесовский подход

- Вероятность — степень уверенности (belief).
- Параметры — случайные величины с априорным распределением.
- Апостериор: \(P(\theta|D) \propto P(D|\theta)P(\theta)\).
- Кредитные интервалы: интервал, в котором параметр лежит с заданной вероятностью.
- Оценки: MAP, posterior mean.

### Сравнение

| | Frequentist | Bayesian |
|---|-------------|----------|
| Параметры | Фиксированы | Случайные |
| Prior | Не используется | Используется |
| Вывод | p-value, CI | Posterior, Credible interval |
| Вычислительная стоимость | Обычно ниже | Выше (MCMC, VI) |

### Примеры в ML

**MAP ≈ L2-регуляризация:** Для линейной регрессии с Gaussian likelihood и Gaussian prior на веса:  
\(\theta_{\text{MAP}} = \arg\max_\theta \log P(D|\theta) + \log P(\theta)\). При prior \(w \sim \mathcal{N}(0, \sigma^2 I)\) добавка к loss — член \(\lambda \|w\|^2\) с \(\lambda \propto 1/\sigma^2\). MAP с гауссовым prior эквивалентен ridge-регуляризации.

**Online Bayes:** При поступлении данных по одному обновление posterior: \(P(\theta|D_1, \ldots, D_t) \propto P(D_t|\theta)\, P(\theta|D_1, \ldots, D_{t-1})\). Posterior на шаге t−1 становится prior для шага t. Используется в Bayesian bandits, streaming learning.
