# Множественное тестирование и достаточные статистики

**Вопрос (EN)**: 10K stocks, 24 price points each. 10K×9999/2 correlations. One pair has r>0.8. Probability by chance? How avoid accidental patterns? Sufficient statistics and Information Bottleneck in ML?

## Ответ

### Вероятность случайной высокой корреляции

При ~50M парах и уровне 0.05 ожидается ~2.5M «значимых» случайных корреляций. Один r>0.8 при таком множестве сравнений вполне вероятен. Curse of big data: много тестов → много ложных открытий.

### Как избежать

- **Коррекция на множественность** (Bonferroni, FDR, Holm).
- **Пере permutation test** — перестановки меток для оценки null.
- **Байесовские методы**, regularization.
- **Валидация** на удержанной выборке или новых данных.

### Числовой пример Benjamini–Hochberg (FDR)

Порядок: 1) сортировать p-values по возрастанию; 2) найти максимальное k, при котором p₍ₖ₎ ≤ k·α/m; 3) отклонить гипотезы 1,…,k.

**Пример:** m=5 тестов, α=0.05. p-values: 0.001, 0.008, 0.02, 0.04, 0.07 (уже отсортированы).

| k | p₍ₖ₎ | k·α/m = k·0.01 |
|---|---|---|
| 1 | 0.001 | 0.01 ✓ |
| 2 | 0.008 | 0.02 ✓ |
| 3 | 0.02 | 0.03 ✓ |
| 4 | 0.04 | 0.04 ✓ |
| 5 | 0.07 | 0.05 ✗ |

Максимальное k с p₍ₖ₎ ≤ k·0.01 — это k=4. Отклоняем первые 4 гипотезы. Bonferroni отклонил бы только первые 2 (порог 0.01).

### Sufficient statistics в ML

Достаточная статистика \(T(X)\) содержит всю информацию о параметре θ: \(P(X|T)\) не зависит от θ. Примеры: \(\bar{x}\) для среднего нормального, \((\sum x_i, \sum x_i^2)\) для μ и σ². Используются в экспоненциальных семействах, EM, сжатии данных.

### Information Bottleneck

**Формулировка:** T — представление (латентное) входа X, Y — целевая переменная. Задача:
\[
\min_{p(t|x)} I(X;T) - \beta\, I(T;Y)
\]
или эквивалентно: минимизировать I(X;T) при ограничении I(T;Y) ≥ const. Параметр β контролирует компромисс сжатия и предсказательной способности.

**Интерпретация:** T должен сохранять информацию о Y, но отбрасывать лишнее из X. Оптимальное представление — minimal sufficient statistic для Y.

**Связь с ML:**
- **VAE:** ELBO ≈ −I(X;T) + β·I(T;Y) (с точностью до членов); β-VAE явно вводит IB-регуляризацию.
- **Сжатие признаков:** bottleneck в autoencoder реализует ограничение I(X;T); при обучении с предсказательной целью получаем IB-оптимизацию.
