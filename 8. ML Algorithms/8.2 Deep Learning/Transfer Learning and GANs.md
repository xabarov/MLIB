# Transfer Learning и GAN

**Вопросы (EN)**: How to build a sentiment classifier with only 1000 labeled tweets? What's gradual unfreezing? Why are GANs hard to train? What do GANs converge to?

## Transfer Learning

### Задача: классификатор тональности с 1000 примерами

1. **Взять предобученную языковую модель** (BERT, GPT-2, RoBERTa) — она уже обучена на огромных текстовых корпусах.
2. **Fine-tune** на 1000 размеченных твитов: заморозить нижние слои (общие признаки), дообучить верхние + классификационную голову.
3. **Data augmentation**: перефразирование, back-translation, synonym replacement.
4. **Градуальное размораживание** (gradual unfreezing) при необходимости.

### Gradual unfreezing

Техника fine-tuning (ULMFiT): модель размораживается слой за слоем, начиная с верхних.

1. Обучаем только классификационную голову.
2. Размораживаем последний трансформер-блок → обучаем.
3. Размораживаем следующий → обучаем.
4. Повторяем до полного размораживания.

**Зачем:** нижние слои содержат общие языковые знания, которые не нужно разрушать. Постепенное размораживание + learning rate decay по слоям (discriminative fine-tuning) снижает риск catastrophic forgetting.

## GANs

### Почему GAN сложно обучать

1. **Нестабильность**: генератор и дискриминатор учатся одновременно — динамика minimax может осциллировать или расходиться.
2. **Mode collapse**: генератор находит небольшое подмножество реалистичных примеров и перестаёт исследовать распределение.
3. **Несбалансированность**: если дискриминатор слишком хорош → градиенты генератора исчезают; слишком плох → генератор получает бесполезный сигнал.
4. **Отсутствие метрики**: потери GAN не коррелируют напрямую с качеством генерации.

### К чему сходится GAN

В теоретическом оптимуме (при неограниченной мощности сетей и достаточном обучении):
- Дискриминатор выдаёт \(D(x) = 0.5\) для всех \(x\) — не может отличить реальные данные от сгенерированных.
- Генератор воспроизводит **истинное распределение данных**: \(p_g = p_{data}\).
- Это Nash equilibrium между генератором и дискриминатором.

На практике точная сходимость редка; современные методы (WGAN, StyleGAN) улучшают стабильность через лучшие loss-функции и архитектурные решения.
