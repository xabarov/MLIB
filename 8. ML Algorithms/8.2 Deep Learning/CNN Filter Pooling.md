# Свёртки, фильтры и пулинг

**Вопросы (EN)**: How filter size affects accuracy and efficiency? How to choose filter size? Locally connected? 1x1 conv? Max vs average pooling? Zero padding?

## Ответ

### Размер фильтра

- **Большие фильтры** — больше рецептивное поле, но больше параметров и вычислений; риск переобучения.
- **Маленькие** — меньше параметров, эффективнее; типично 3×3, каскад для рецептивного поля.
- **Выбор:** 3×3 как базовый вариант; комбинации (3×3, 5×5) в Inception; 1×1 для сжатия каналов.

### Locally connected

Свёртка — локально связный слой с разделением весов (weight sharing). Каждый нейрон связан только с локальной областью входа.

### 1×1 свёртка

Операция по каналам: смешивание каналов без изменения пространственных размеров. Уменьшает число каналов, добавляет нелинейность (с ReLU), снижает параметры.

### Max vs Average pooling

- **Max** — выделяет наиболее выраженные признаки; типичен для классификации.
- **Average** — сглаживает; подходит для снижения размерности при сохранении общей картины.

### Zero padding

Сохраняет пространственные размеры после свёртки; контролирует размер выхода. «Same» padding — выход того же размера, что и вход.
