# L1-регуляризация (Lasso)

**Вопрос (EN)**: Why does L1 regularization tend to lead to sparsity while L2 regularization pushes weights closer to 0?

## Ответ

### Норма L1

Норма L1 (манхэттенская) — сумма абсолютных значений компонент вектора:
\[
\|w\|_1 = \sum_{i=1}^{n} |w_i|
\]
Для вектора [2, 4]: \(|2| + |4| = 6\).

### L1-регуляризация в функции потерь

\[
L_{\text{total}} = L + \lambda \sum_i |w_i|
\]
где \(w_i\) — веса модели, \(\lambda\) — коэффициент регуляризации.

### Отрицательные веса

Для \([-1, -2.5]\): \(|{-1}| + |{-2.5}| = 3.5\). Модуль даёт одинаковый вклад для положительных и отрицательных весов.

### Почему L1 даёт разреженность

Производная L1 по \(w_i\): \(\frac{\partial}{\partial w_i} \lambda|w_i| = \lambda \cdot \text{sign}(w_i)\) — либо \(+\lambda\), либо \(-\lambda\); в нуле не определена.

Градиент по модулю постоянный при любом ненулевом \(w_i\). В сочетании с колебаниями от основного loss веса стремятся к нулю и могут его достичь — получается разреженная модель.

### Производная L2

\(\frac{\partial}{\partial w_i} \lambda w_i^2 = 2\lambda w_i\). При \(w_i \to 0\) градиент стремится к нулю, вес медленно приближается к нулю, но обычно не обнуляется. L2 сжимает веса, но не делает их строго нулевыми.

### Недостатки Lasso

1. **Большая размерность (p > n)** — L1 выбирает не более n переменных; при «коротких» данных ограниченная отборность.
2. **Избыточная разреженность** — если сигнал распределён по многим переменным, L1 может удалить полезные.
3. **Эластичная сеть** — часто лучше сочетание L1 + L2 (Elastic Net).
